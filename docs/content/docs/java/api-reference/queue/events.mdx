|---|---|

## Webhooks & autoscaler

`WebhookManager.attach(taskito)` registers outcome webhooks, and
`Scaler.start(taskito, ScalerOptions.onPort(9090))` serves a queue-depth
endpoint for external autoscalers — both build on this client.
