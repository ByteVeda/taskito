## Workers

| Method | Description |
|---|---|
| `worker()` → `Worker.Builder` | Begin building a [worker](/java/api-reference/worker). |
| `listWorkers()` → `List<WorkerInfo>` | Registered workers (heartbeat + identity). |

## Middleware

| Method | Description |
|---|---|
| `use(Middleware)` | Cross-cutting hooks — `onEnqueue`, `before`/`after`/`onError`, and the outcome hooks. |

## Circuit breakers

| Method | Description |
|---|---|
| `listCircuitBreakers()` → `List<CircuitBreakerState>` | Live state of every registered [circuit breaker](/java/api-reference/task#circuitbreakerconfig) — open/closed/half-open, failure counts, next-probe time. |

## Workflows

| Method | Description |
|---|---|
| `submitWorkflow(Workflow)` / `submitWorkflow(Workflow, Map)` | Submit a [workflow](/java/api-reference/workflows) → `WorkflowRun`. |
| `workflowStatus(runId)` / `cancelWorkflow(runId)` | Run queries and cancellation. |
