# Workflows

taskito provides two workflow models. See the dedicated **[Workflows section](../../workflows/index.md)** for full documentation.

- **[DAG Workflows](../../workflows/building.md)** — Multi-step pipelines as directed acyclic graphs with fan-out, conditions, gates, sub-workflows, incremental caching, and visualization.
- **[Canvas Primitives](../../workflows/canvas.md)** — Lightweight chain, group, and chord composition for simple pipelines.
